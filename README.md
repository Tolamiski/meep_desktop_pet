# meep_desktop_pet
一只在电脑桌面上的小丘鹬！
An American Woodcock on your desktop!

# 操作指南 / How to play	           
🖱️ 鼠标左键点击 / Left Click	    触发动作动画 / Trigger "meep!"
🖱️ 鼠标左键按住拖动 / Left Drag	自由移动宠物位置 / Move woodcock freely
🖱️ 鼠标移入窗口 / Hover Over	    触发动作动画 / Trigger "meep!"
🖱️ 鼠标右键点击 / Right Click	  弹出退出确认对话框 / Show exit confirmation dialog
✅ 点击"是" / Click "Yes"	      退出程序 / Exit 
❌ 点击"否" / Click "No"	      继续运行 / Keep running
